# Derivco-mzansibuilds
MzansiBuilds: A platform for developers to build in public, share milestones, and collaborate. Developed for the Derivco Code Skills Quest.
This project was developed as a submission for the **Derivco Code Skills Quest** (Stage 2).
Features:
This application satisfies the core requirements of the Derivco assessment:
* User Authentication: Secure account creation and profile management for developers.
* Project Management: Create new project entries, define current stages (Idea, MVP, Beta), and request specific support.
* Live Feed & Collaboration: A central hub to view what other developers are building, with the ability to comment and "raise a hand" to collaborate.
* Milestone Tracking: Continuous progress updates for active projects.
* Celebration Wall: A dedicated showcase for completed developer projects.

## 💻 Tech Stack
* Framework: ASP.NET Core MVC / Web API (Update this if you are using Blazor or something else)
* Language: C#
* Database: Microsoft SQL Server & Entity Framework Core
* Frontend: HTML, CSS, javascript,Bootstrap

## Design Theme
As per the assessment guidelines, the UI strictly follows a **Green, White, and Black** color palette to ensure a clean, accessible, and professional user experience.
